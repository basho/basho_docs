List available HTTP resources for the Riak node. This can be used by clients to
automatically recognize the location of the resources for specific operations.

The standard resources are:

* `riak_kv_wm_buckets` - [Bucket Operations]({{<baseurl>}}riak/kv/3.2.0/developing/api/http/#bucket-operations)
* `riak_kv_wm_index` - [HTTP Secondary Indexes]({{<baseurl>}}riak/kv/3.2.0/developing/api/http/secondary-indexes)
* `riak_kv_wm_link_walker` - [HTTP Link Walking]({{<baseurl>}}riak/kv/3.2.0/developing/api/http/link-walking)
* `riak_kv_wm_mapred` - [HTTP MapReduce]({{<baseurl>}}riak/kv/3.2.0/developing/api/http/mapreduce)
* `riak_kv_wm_object`- [Object/Key Operations]({{<baseurl>}}riak/kv/3.2.0/developing/api/http/#object-key-operations)
* `riak_kv_wm_ping` - [HTTP Ping]({{<baseurl>}}riak/kv/3.2.0/developing/api/http/ping)
* `riak_kv_wm_props` - [HTTP Set Bucket Properties]({{<baseurl>}}riak/kv/3.2.0/developing/api/http/set-bucket-props)
* `riak_kv_wm_stats` - [HTTP Status]({{<baseurl>}}riak/kv/3.2.0/developing/api/http/status)

## Request

```bash
GET /
```

Headers:

* `Accept` - `application/json` or `text/html`

## Response

Normal status codes:

* `200 OK`

Important headers:

* `Link` - all resources that are described in the response body, but in Link
form
