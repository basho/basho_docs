Check if the server is alive

## Request

Just the RpbPingReq message code. No request message defined.

## Response

Just the RpbPingResp message code. No response message defined.

## Example

Request

```bash
Hex    00 00 00 01 01
Erlang <<0,0,0,1,1>>
```

Response

```bash
Hex    00 00 00 01 02
Erlang <<0,0,0,1,2>>
```
